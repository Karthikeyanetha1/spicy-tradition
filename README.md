## 📌 Overview

Spicy Tradition is a full-featured restaurant website built as a **CodeWithK** freelance demo. It captures the warmth of authentic Telangana food culture with a Playfair Display serif aesthetic, Pixabay food photography, Font Awesome 6 icons, and smooth CSS animations throughout.

> 🔗 **Live:** [spicy-tradition.vercel.app](https://spicy-tradition.vercel.app)

---

## ✨ Features

| # | Feature | Description |
|:---:|---|---|
| 🍽️ | **Filterable Menu** | All · Main Course · Starters · Biryani · Desserts |
| 📢 | **Marquee Ticker** | Scrolling dish names across saffron banner |
| 📖 | **Story Timeline** | Founding story 2012 → 2024 with milestone events |
| 🌟 | **Weekend Specials** | 4 chef's special cards with WhatsApp order links |
| ⭐ | **Testimonials** | 3 verified Zomato/Google/Swiggy reviews |
| 🕐 | **Smart Hours** | JS auto-detects today and shows correct hours |
| 📬 | **Contact Form** | EmailJS — styled glass form with icons on each label |
| 📱 | **Mobile Toggle** | Hamburger menu with drawer animation + body lock |
| 💬 | **WhatsApp CTAs** | Direct order/reservation links throughout |
| 🎨 | **Font Awesome 6** | 96 icons — all UI elements, social links, labels |
| ✨ | **11 Keyframes** | fadeUp, slideRight, float, pulse, marquee + more |

---

## 🛠️ Tech Stack

```
Frontend       →   HTML5, CSS3, Vanilla JavaScript
Icons          →   Font Awesome 6.5 CDN (96 icons used)
Images         →   Pixabay CDN — free food photography, no hotlink block
Email          →   EmailJS SDK — real delivery, no backend
Fonts          →   Google Fonts — Playfair Display + Lato + Dancing Script
Animations     →   11 custom CSS @keyframes
Hosting        →   Vercel (auto-deploy from GitHub)
```

## 🍔 Hamburger Toggle

```javascript
hamburger.addEventListener('click', () => {
  hamburger.classList.toggle('open');     // Animates bars → X
  mobileMenu.classList.toggle('open');   // Slides drawer down
  document.body.style.overflow = ...;    // Locks scroll when open
});
// Also closes on outside click & nav link click
```
